# 📥 Install Repository

## 🔗 Repository URL

```txt
https://raw.githubusercontent.com/sad25kag/BetbetMiro-Extension/master/repo.json
```

---

## 📲 Cara Install

1. Buka **CloudStream**
2. Masuk ke **Settings**
3. Pilih **Extensions**
4. Klik **Add Repository**
5. Paste link repository di atas
6. Install repository
7. Selesai 🎉

# 🛠️ Build From Source

## Clone Repository

```bash
git clone https://github.com/sad25kag/BetbetMiro-Extension.git
```

## Masuk Folder

```bash
cd BetbetMiro-Extension
```

## Build Extension

```bash
./gradlew make
```

## Output Build

```txt
/builds
```
